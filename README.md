# Hồ Sơ 0 Giờ — Bản Nhạc Lúc 0 Giờ

Game trinh thám 2D nguyên bản chơi trực tiếp trong trình duyệt. Người chơi phải ghim lời khai, khám nghiệm vật chứng, tự xếp dòng thời gian và dựng đủ năm mắt xích trước khi buộc tội.

- [Chơi bản web](https://conan1.vercel.app)
- [Mã nguồn và hướng dẫn](./Conan_Ban_Nhac_Luc_0_Gio_Local/conan-midnight-case/README.md)

Phiên bản này đã loại bỏ tên, nhân vật và dịch vụ phụ thuộc vào thương hiệu trinh thám có sẵn. Nội dung, hình ảnh và mã nguồn thuộc dự án gốc **Hồ Sơ 0 Giờ**.
